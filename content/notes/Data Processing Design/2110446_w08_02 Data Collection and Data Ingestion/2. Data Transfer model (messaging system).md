# Point-to-point

ตัว source จะส่งข้อมูลไปหา receiver โดยตรง ทำงานแบบ synchronous

# Publish-subscribe

ตัว source/producer จะ publish event ไปหา topic และ receiver/consumer ที่ subscribe จะได้ข้อมูลมาจาก topic อีกที ทำงานแบบ asynchronous (เพราะเราไม่รู้ว่า producer จะใช้เวลา publish นานแค่ไหน และไม่รู้ว่า consumer จะได้รับ message เมื่อไหร่)
