# Apache Kafka คืออะไร?

Apache Kafka คือ distributed streaming platform ใช้สำหรับ real-time data pipelines and applications ที่พัฒนาโดย LinkedIn
- Publish/Subscribe messaging system
- Real-time
- Reliable Data Delivery
- Scalable and Fault-tolerance

## Kafka Basic Architecture

![[2110446_w08_03_1_01 Kafka Basic Architecture.png]]

Components:
1. Producer: source ที่ส่งข้อมูลไปเก็บไว้ที่ Topic
2. Topic: ถีงรวมข้อมูลจาก Producer ที่รับข้อมูลตามลำดับ (reliable order บน partition)
3. Consumer: ตัวรับข้อมูลจาก Topic ที่ subscribe เอาไว้
