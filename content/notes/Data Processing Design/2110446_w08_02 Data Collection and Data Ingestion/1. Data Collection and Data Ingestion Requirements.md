สมมติมี Smart Meter เครื่องหนึ่ง
- วัดปริมาณไฟฟ้าที่ใช้
- ส่งข้อมูลทุก 15 นาที
- ในไทยมีอยู่ 20 ล้านครัวเรือน
- หมายความว่ามี event เข้ามา 22222 events/sec
- ถ้าใน 1 event มีประมาณ 1k data (power consumption, temperature)
- total data = 80GB/hr. or 1.9TB/day or >700TB/year

เพียง meter เครื่องเดียวเราจะเห็นปัญหาที่จะเจอได้หลายอย่าง
1. Variety of data sources and destinations: มีข้อมูลหลากหลายประเภท แลถอาจจะต้องดึงจากหลายแหล่ง และจัดเก็บหลายแหล่ง (to database, as files, to service)
2. Complex data flow: จะนำข้อมูลจาก 1/many sources ไป 1/many destinations
3. Reliable data transfer: ข้อมูลจะมาเสมอหรือไม่ จะไม่มีสูญหายระหว่างทางรึเปล่า
4. In-flight data processing: การทำ data cleansing, data standardization, data fusion
5. Scalability and Fault-tolerance: ขยายได้ และทนต่อความผิดพร่อง
