Data Pipeline คือ กระบวนการ process data ตั้งแต่เก็บข้อมูลดิบ (raw data) ไปจนถึงนำเสนอ (Presentation) มี 5 ขั้นตอน
1. Collection: การได้รับข้อมูลจากแหล่งข้อมูลต่าง ๆ (Sensor, CCTV, Thermometer, Sound)
2. Ingestion: การนำข้อมูลที่ได้มาจัดเก็บ ต้องคำนึงว่า จะเชื่อมต่อจากไหนไปไหน จำนวนและคึวามถี่ของ source
3. Preparation: Cleansing, Transformation, Handle missing data, Standardization (data format), Data Wrangling
4. Computation: การประมวลผลข้อมูล OLAP: Online Analytical Processing คือการวิเคราะห์เชิงหลายมิติ (multidimensional format) เช่น operational report, machine learning, prediction model
5. Presentation: นำเสนอข้อมูลออกมาใน graph, dashboard
